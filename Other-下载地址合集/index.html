Environment

  • Python（Python环境，建议开加速，不然下一年）
  • Git（Github客户端）
  • Nodejs
  • JAVA
  • SQL Server 2019（在里面的服务器里，自己找找吧）
  • mingw-w64(C语言编译器)
  • Anaconda（Python多环境）
  • Qt（图形界面开发）

IDE

  • VS Code(集成环境，我主要用来写markdown)
  • Visual Studio(C++编译集成环境)
  • Pycharm(Python集成编译环境)
  • HBuilder X（html编辑器）
  • Dev-C++（C++的IDE）

Softwares

  • QQ(PC版QQ官网)
  • 网易云音乐（精神毒药）
  • 钉钉（日你妈钉钉）

Tools

  • 7Z（压缩包工具）
  • OBS（录屏工具）
  • 迅雷（下载工具）
  • nn加速器（免费的加速器）
  • SSMS（数据库管理工具）
  • PhpStudy（本地web环境搭建）
  • VMware（虚拟机）
  • 火绒安全（杀毒软件、启动项管理、右键管理）
  • StartIsBack（美化开始窗口、任务栏居中）
  • 腾讯桌面管家（桌面文件美化）

Game

  • Steam(Steam平台)

Theme

  • 鼠标主题（Mac主题的鼠标）

Microsoft Store

微软商店内下载的，Windwos自带的微软商店！直接搜索就能搜到

  • HyPlayer（第三方轻量级网易云播放器，个人感觉音质好像不太一样）
  • TranslucentTB(汉化 by tpxxn)（桌面汉化）
